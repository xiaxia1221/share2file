# FileShareSaver (分享保存器)

这是一个极简的安卓应用，用于通过系统分享接口快速保存文件，并集成到系统文件选择器（上传接口）中。

## 核心功能

1.  **自动接承分享**：支持从微信、浏览器等应用分享文件到本项目。
2.  **静默保存**：文件自动保存到应用私有目录，不要求手机存储权限。
3.  **自动命名**：同名文件自动追加序号，如 `test.pdf` -> `test (1).pdf`。
4.  **文件选择器集成**：在浏览器或应用进行“文件上传”时，可以在系统文件选择器的侧边栏找到“分享保存器”，直接选择之前保存的文件。

## 技术实现

-   **ShareReceiverActivity**: 透明 Activity，负责接收 `ACTION_SEND` intent 并调用保存逻辑。
-   **FileStore**: 处理文件流写入与命名冲突逻辑。
-   **SavedDocumentsProvider**: 继承 `DocumentsProvider`，将应用私有目录中的文件暴露给系统的存储访问框架（SAF）。

## 使用说明

1.  编译并安装 APK。
2.  当你在其他应用（如微信）看到想要保存的文件时，点击“分享”或“用其他应用打开”，选择“分享保存器”。
3.  系统弹出 Toast 提示保存成功。
4.  当你需要在浏览器上传文件时，在弹出的文件选择器中，点击左侧菜单选择“已保存的分享文件”，即可看到所有通过本软件保存的文件，按日期排序显示。
